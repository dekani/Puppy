  2. Lancer le script "Remaster CD", se trouvant dans le menu "Menu -> Configurer -> Remaster Puppy live-CD".

Vérification orthographique

Seamonkey Composer (l'éditeur de HTML) a un vérificateur d'orthographe, mais certaines versions de PUPPY/TOUTOU n'ont aucun dictionnaire (ou seulement le dictionnaire EN/EU). C'est très facile d'y remédier. Le site Web pour les dictionnaires dans toutes les langues est : http://dictionaries.mozdev.org/installation.html.
Et vous pouvez utiliser soit un navigateur pour aller directement à cette adresse, soit ouvrir le correcteur orthographique dans Composer, par "Edit -> Orthographe..." clic sur  "Télécharger plus". . Les instructions sur le site Web indiquent clic gauche pour télécharger et installer un dictionnaire, cependant moi j'ai eu : "erreur de téléchargement". Donc à la place : clic droit et choisir télécharger et enregistrer le fichier, par exemple "spell-fr.FR.xpi". Puis lancer Seamonkey, choisir "Fichier - > Ouvrir ..." et le fichier ".xpi" s'installera automatiquement .
Les dictionnaires téléchargés sont dans : /usr/lib/mozilla/components/myspell/.

Flash

PUPPY/TOUTOU a le plugin Adobe Flash player, se trouvant dans /usr/lib/mozilla/plugins. SeaMonkey, Firefox et Opera sont capable de le trouver dans ce fichier.

Actuellement (avril 2008), il y a des problèmes d'affichage plein-écran avec quelques vidéos depuis youtube.com. Une version plus récente du plugin peut marcher pour vous.
Il y a des discussions sur le forum de PUPPY à ce sujet, avec quelques suggestions.

Pour installer un nouveau plugin Flash, télécharger le fichier depuis Adobe, décompresser-le (juste cliquer dessus), et copier 'libflashplayer.so' vers /usr/lib/mozilla/plugins.
Puis fermer et redémarrer le navigateur.

Codecs audio/vidéos

XINE est la bibliothèque multimédia utilisée par PUPPY/TOUTOU et Gxine est une interface graphique pour le lecteur. Gxine a aussi plugin pour le web, compatible avec SeaMonkey, Firefox et Opera.

Noter que vous trouverez le plugin de Gxine dans /usr/lib/mozilla/plugins. SeaMonkey, Firefox et Opera le trouveront dans ce dossier.

Gxine a lui aussi quelques problèmes de lecture en mode plein-écran, mais il y a une solution facile. D'abord, maximiser la fenêtre (l'icône du milieu, en haut à droite, dans la fenêtre) , puis choisir 'Vue -> mode plein-écran' dans le menu.

La bibliothèque XINE dans PUPPY/TOUTOU peut lire la plupart des fichiers audio/vidéo, mais si l'un d'entr'eux, ne peut être lu, il y a un paquet .PET avec des 'extra-codecs' (des plugins pour le support de plus nombreux formats de fichiers).
Le meilleur endroit pour trouver des infos complémentaires, la page HOWTO: multimedia.
