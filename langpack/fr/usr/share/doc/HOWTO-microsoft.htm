HOWTO : Documents Microsoft

Nous savons que Microsoft Office -- est defacto la norme, nous devons donc en tenir compte. Le problème le plus fréquent, les documents de Word -- De nombreux hommes d'affaires taperont un document en utilisant Word, l'attacheront ensuite à un courrier électronique et s'attendront à ce que les destinataires soient capables de le voir.
Donc, voyons pour Word ...

Fichiers MS Word .doc

Microsoft utilise un format propriétaire et peu documenté pour les fichiers Word et pour aggraver les choses, un document préparé avec Word2000 ne peut être lu avec Word97. Cela enferme les clients dans le cercle vicieux des mises à jour de Microsoft.

De toute façon, concentrons-nous sur la situation dans PUPPY/TOUTOU.

Jusqu'à la version 0.9.4 incluse, PUPPY avait un programme nommé Antiword, qui convertissait les fichiers MS WORD DOC en Postscript, ils pouvaient ainsi être vus mais pas imprimés

Mais depuis la version 0.9.5, PUPPY comprend Abiword traitement de textes, dans lequel est intégré wv.Ceci autorise l'import/export des fichiers MS DOC par Abiword.

La page d'aide de Abiword : /usr/share/doc/abiword.htm

Noter aussi que le gestionnaire de fichiers Rox, le client-mail Sylpheed et d'autres applications sont configurés pour lancer Abiword. Un clic gauche sur un fichier DOC dans Rox, et le fichier s'ouvrira dans Abiword.

Le fichier de configuration de ROX est dans /root/Choices/
Celui de Sylpheed est dans /root/.sylpheed/sylpheedrc
Les types de MIME et les applications externes rattachés à Sylpheed sont liés au contenu de /etc/mimetypes et /etc/mailcap/
Seamonkey lit aussi en /etc/mimetypes et /etc/mailcap, afin de permettre lecture/affichage d'un fichier que vous êtes sur le point de télécharger.

Fichiers MS Word .rtf

RTF signifie Rich Text Format, et c'est un format de fichier pour les programmes de traitement de textes. La norme RTF est abondamment documentée et tous les traitements de textes importants peuvent l'ouvrir et l'enregistrer, MS WORD inclus. Ainsi, les utilisateurs de MS Word devraient être encouragés à sauvegarder des documents au format RTF, avant l'envoi de ceux-ci à d'autres personnes.

Jusqu'à la version 0.9.4, PUPPY avait Ted comme traitement de textes, dont le format par défaut est RTF.

Depuis la version 0.9.5, Abiword peut importer/exporter le RTF.

Et une fois encore, ROX-Filer et Sylpheed ont été configurés pour reconnaître l'extension ".rtf". Dans le cas de ROX, un clic gauche suffit.

Fichiers du tableur MS Excel

Le tableur Excel est une composante de la suite Office.

Depuis la version 0.9.7 PUPPY, contient ABS, un tableur aussi. ABS est toujours disonible en paquet PET. Le format par défaut d' ABS peut être converti en format Excel, et vice versa, au moyen d'une macro. La documentation se trouve dans le paquet supplémentaire PET.

PUPPY depuis la version 0.9.8 et suivantes a Planmaker Free Edition, qui est une version gratuite de Planmaker, un produit commercial. L'édition gratuite n'est pas bridée et n'a aucune restriction d'utilisation, sauf la limitation du nombre maximal de rangées/colonnes pouvant être éditées. Les vraiment bonnes nouvelles sont qu'il peut ouvrir, éditer et sauver les fichiers du format d'Excel. Planmaker est toujours disponible comme un paquet PET.

La documentation est en format PDF : planmaker.pdf (seulement si le paquet Planmaker est installé)
La licence : license.txt (seulement si le paquet Planmaker est installé)

PUPPY version 1.0.4 et supérieure, intègre Gnumeric, doté d'un jeu complet de plugins, dont l'import/export du format d'Excel. Gnumeric est open source, gratuit et vraiment bon.

Finalement, si vous voulez le nec plus ultra, OpenOffice est disponible pour PUPPY/TOUTOU, pas dans le liveCD mais sous forme d'un paquet PET ou d'un fichier SFS. Ce dernier est recommandé. Si vous ne savez pas ce qu'est un fichier SFS, rendez-vous sur cette page : http://puppylinux.com/development/package-management.htm.

Polices MS

Oui, les polices MS TrueType sont facilement utilisables dans PUPPY/TOUTOU -- c'est juste une question de duplication (ou bien, si c'est un problème légal, un lien symbolique peut être créé depuis la partition PUPPY/TOUTOU vers les polices de la partition Windows).

La page d'aide de Abiword explique comment faire : /usr/share/doc/abiword.htm

Open Document Format

Je le mentionne ici, ODF devient une norme internationale de format de fichier de document. Abiword peut maintenant importer et exporter des fichiers ODF.

Ouais, mais en ce qui concerne MS Office ? Microsoft a été fortement opposé à ODF et continue de le dénigrer. Il n'est pas dans l'intérêt de Microsoft d'avoir des normes de document ouvert, car en ayant des formats de fichiers fermés, il peut toujours avoir une longueur d'avance sur la concurrence. Ainsi, MS ne propose pas l'import/export d'ODF dans OFFICE, cependant la bonne nouvelle : une société a développé un plugin qui le permet et c'est gratuit.

Je n'ai pourtant pas de détails sur où on peut se procurer ce plugin 0DF pour MS Office.

Noter : vous pouvez trouver une série d'exemples de fichiers dans /usr/share/examples, fichiers .xls (Excel) et .doc (Word) inclus.
